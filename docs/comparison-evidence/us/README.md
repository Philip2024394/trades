# USA Comparison Evidence Files

_Per-competitor evidence for `/trade-off/us/compare-platforms/`. Required before public publish per `docs/LEGAL_US_COMPARATIVE_ADVERTISING.md`._

**Why this matters:** Lanham Act §43(a) creates competitor standing to sue for false-advertising claims. Contemporaneous evidence is the substantiation defence. §35(a) fee-shifting in "exceptional" cases means stale pricing = attorney's-fees exposure.

## USA-specific evidence rules

- **Pricing is highest-risk cell** — Angi lead pricing is dynamic (varies by trade, zip, competition), Thumbtack varies constantly. Screenshot the pricing page + note the exact zip/trade filter used.
- **Dated screenshot required** for every price cell. PNG only (lossless, tamper-evident).
- **Filename convention**: `pricing-{YYYY-MM-DD}.png`, `features-{YYYY-MM-DD}.png`, etc.
- **Legal hold** — if a competitor changes their page after you scored them, keep the old screenshot as proof of what was true when you scored.
- **State-specific pricing** — some USA competitors have per-state pricing. Screenshot the highest AND lowest tier if you can find both.

## How to keep this fresh

1. Run `node scripts/scan-blueprint.mjs` after every meaningful change.
2. When you add a new app / lib / platform area, add a `README.md` to its folder with a 1-line summary in the second paragraph.
3. When you add a new page / API route / migration, put a `//` or `--` leading comment describing what it does.
4. The scanner picks all of that up automatically — this doc regenerates in ~2 seconds.

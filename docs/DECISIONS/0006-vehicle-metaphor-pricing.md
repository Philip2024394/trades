# ADR-0006: Vehicle metaphor for pricing tiers

Status: Accepted
Date: 2026-07-13

## Context

The packages page needed a way to communicate the 4-tier ladder (Free / Canteen / Marketplace / The Works) at a glance. Generic pricing-page icons (checkmarks, stars, sparkles) feel corporate and don't resonate with the trades audience. The AI-star / sparkles icon set was explicitly ruled out ("crap").

## Decision

Each tier's hero image is a vehicle that maps to the merchant's progression:
- **Free** = push bike (start moving)
- **Canteen** = motor bike (step up)
- **Marketplace** = van (load up — the working trades vehicle)
- **The Works** = jeep (all-terrain — everything unlocked)

Positioning copy leans into the metaphor: "Start moving" / "Step up. Get seen." / "Load up. Get selling." / "All terrain. Everything unlocked." Star count beside each tier name (1 / 2 / 3 / 4 filled yellow stars) reinforces the ladder without needing to read the price.

## Consequences

- **Positive:** Trades-native language. Every UK tradesperson thinks in vehicles — the van IS the business.
- **Positive:** Instant ladder recognition without reading. Bike → van → jeep = obvious progression.
- **Positive:** Memorable and distinctive. Most SaaS pricing pages look identical; this one doesn't.
- **Positive:** Van as Marketplace is on-point — merchants who sell products already brand their vans; this connects directly.
- **Negative:** Copywriting depends on the metaphor holding up. If we later add a tier between two existing ones, the vehicle line-up has to expand cleanly (e.g. adding a "Pickup" between Van and Jeep).

## Alternatives considered

- **Generic Lucide icons** (UserRound / Home / ShoppingBag / Package) — rejected as forgettable. This is what the page had initially; it worked but didn't sell.
- **Numbered tiers only** (Tier 1 / 2 / 3 / 4) — rejected as sterile.
- **Named after trades tools** (Toolbox / Van Kit / Site Rig / Master Toolkit) — considered but too abstract vs. the vehicle progression.
